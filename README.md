# TODO app

This is a cool TODO list app created with Node.

---

## Requirements

For development, you will only need Node.js and a node global package installed in your environement.

## Install

    $ git clone https://github.com/Dea10/node_todo_app.git
    $ cd node_todo_app
    $ npm install

## Running the project

    $ node app.js

Enjoy :)
